Smart Food Decision Assistant 🍽️

A lightweight web application that helps users make smarter food choices in real-time based on their recent eating habits, time gaps, and personal health conditions.

🚀 Features:

* Quick meal logging (Heavy, Light, Junk, Skipped)
* Personalized health setup (Diabetes, BP, Weight Loss, etc.)
* Smart food suggestions based on user behavior
* Balance Mode (suggests healthy combos with unhealthy choices)
* Food health scoring system (Healthy / Balanced / Unhealthy)
* Simple habit insights (e.g., frequent junk consumption)
* Indian food context (Biryani, Dosa, Thali, etc.)

🧠 How it Works:
The app uses simple rule-based logic to analyze:

* Last meal type
* Time since last meal
* User health preferences

Based on this, it provides real-time recommendations and nudges users towards healthier or more balanced eating decisions.

💡 Key Idea:
Instead of restricting users, the app promotes a balanced diet by suggesting healthier combinations alongside preferred food choices.

🛠️ Tech Stack:

* JavaScript (Vanilla)

☁️ Deployment:
Hosted on Google Cloud using static website hosting.

Built as part of a challenge to demonstrate behavior-based decision-making and personalized user experience in a simple, lightweight application.
